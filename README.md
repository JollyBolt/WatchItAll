Watch It All is a dynamic and responsive web application developed using React.js, Tailwind CSS, and Material UI. It provides a seamless and engaging user experience for discovering and watching YouTube videos. The application leverages the power of the 'YouTube v3' API from RapidAPI, offering a rich feature set for users to explore, search, and view videos with ease.

## Features

1. <b>Intuitive User Interface:</b> Watch It All boasts an intuitive and user-friendly interface, ensuring an enjoyable browsing experience.

2. <b>Video Search and Discovery:</b> Users can easily search for videos by keywords, explore various categories, and find the content they love.

3. <b>Video Playback:</b> The application offers smooth video playback, with support for full-screen mode and video controls.

4. <b>Responsive Design:</b> Watch It All is fully responsive, making it accessible on various devices, including desktops, tablets, and mobile phones.

5. <b>Theming with Tailwind CSS:</b> The application's aesthetics are enhanced by custom theming using Tailwind CSS, providing a visually pleasing experience.

6. <b>API Integration:</b> The project incorporates the 'YouTube v3' API from RapidAPI, allowing real-time data retrieval and video updates.

## Tech Stack

- <b>React.js:</b> Used for building the frontend of the application, enabling a dynamic and interactive user interface.
- <b>Tailwind CSS:</b> Implemented for responsive and customized styling, enhancing the visual appeal of the app.
- <b>Material UI:</b> Integrated components for a consistent and polished design.
- <b>'YouTube v3' API (RapidAPI):</b> Utilized to fetch video data and ensure up-to-date content.


## Installation

- Install dependencies with npm.

```bash
  npm install 

```
- Create .env file. Add the following variables.  
```bash
  VITE_URL = https://youtube-v31.p.rapidapi.com
  VITE_RAPID_KEY= YOUR_API_KEY
```
- You can get YOUR_API_KEY from https://rapidapi.com/hub. Create your account. Go to Apps. Click on the dropdown of the default application and go to authorization. There you will find your API key.
